红黑树学习笔记

红黑树

红黑树（red-black tree）是一种特殊的平衡二叉树，它有如下5个规则：

  1. 每个节点是红色的或者是黑色的。
  2. 根节点和叶子节点（NIL）是黑色的。
  3. 红色节点的儿子节点都是黑色的。
  4. 每个节点到叶子节点（NIL）的简单路径上的黑色节点个数都是一样的。

由第3、4条规则我们可以知道，最长路径的长度不会超过最短路径的两倍。所以说，红黑树是一种比较平衡标准比较宽松的平衡二叉树。也正是因为这一点，红黑树在进行调整时可以比平衡二叉搜索树进行更少次数的旋转。

定理：包含n个节点的红黑树，高度至多为2log(n+1)

证明方法1（利用完全二叉树的性质）：
假设一棵高度为h的红黑树，其根节点的黑高度bh(root) = m
由规则四可知，其任意一条到叶子节点的路径的黑高度均为m
∴ 其到其任意一条叶子节点路径的高度均大于等于m
即该红黑树至少是一颗高度为m的完全二叉树
∴ 其包含节点个数为n ≥ 2^m^ - 1
即 m ≤ log(n + 1)
由规定三可知，m ≥ h/2
∴ h ≤ 2log(n + 1)，得证。

证明方法2（数学归纳法）：

  1. 当一棵红黑树的子树高度h为1时，即该子树为NIL节点，其黑高度bh(x) = 0, n = 0, n ≥ 2^bh(x)^ -1 成立
  2. 假设：当一颗红黑树的子树高度为x时，该子树的黑高度为bh(x)，n ≥ 2^bh(x)^ - 1成立

对红黑树的任何一个节点，若其左右子节点满足n ≥ 2^bh(x) - 1
则以该节点为根的子树包含的节点数为
2^bh(xl)^ - 1 + 2bh^(xr)^ - 1 + 1
= 2^bh(xl)^+2^bh(xr)^ - 1
∵ bh(xl) ≥ bh(x) - 1，bh(xr) ≥ bh(x) - 1
∴ n ≥ 2^bh(x)-1^ + 2^b(x)01^ - 1 = 2^bh(x)^ -1
即 bh(x) ≤ log(n + 1)
又 bh(x) ≥ h/2，
∴ h ≤ 2log(n + 1)，得证。
